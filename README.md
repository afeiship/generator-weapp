# generator-weapp 
> Yeoman generator for wechat small app (weapp,mina).

## install:
+ install generator:
```bash
git clone git@github.com:afeiship/generator-weapp.git
cd generator-weapp
npm link
```


## usage:
```bash
## app:
yo weapp

## page:
yo weapp:page --d=src/pages

## component:
yo weapp:component --d=src/components/views

## bower_component:
yo weapp:bower_component
```
