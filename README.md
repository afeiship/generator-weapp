# generator-weapp 
> Yeoman generator for wechat small app (weapp,mina).


## usage:
+ install generator:
```bash
npm install -g afeiship/generator-weapp 
```
+ create weapp:
```bash
yo weapp
```
+ yo weapp:page
```bash
yo weap:page --dir=src/pages
```

+ yo weapp:component
```bash
yo weap:page --dir=src/components
```
