# generator-weapp 
> Yeoman generator for wechat small app (weapp,mina).

## install:
+ install generator:
```bash
npm install -g afeiship/generator-weapp 
cd generator-weapp
npm link
```


## usage:
+ create weapp:
```bash
yo weapp
```
+ yo weapp:page
```bash
yo weap:page --d=src/pages
```

+ yo weapp:component
```bash
yo weap:page --d=src/components
```
